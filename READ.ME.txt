# 🌲 Myslivecký spolek – moderní webová prezentace

## 📌 Úvod
Tento projekt je moderní webová prezentace mysliveckého spolku doplněná o jednoduchou interaktivní hru (lovecký simulátor). Web slouží jako fiktivní oficiální stránka spolku a obsahuje informace o členech, akcích, galerii a kontaktech.

🔗 Živá verze (GitHub Pages):  
https://tvuj-github-link.github.io

---

## 🛠 Použité technologie
- HTML5
- CSS3 (Flexbox, Grid, moderní layouty)
- Vanilla JavaScript (ES6+)

# ⚙️ 1. Performance (Výkon)

Performance optimalizace zajišťuje rychlé načítání webu, minimalizaci dat a efektivní vykreslování.

- Použití jednoduchého CSS bez frameworků
- Minimalizace externích knihoven
